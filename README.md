# Uploader

A simple app to share files without store them in server.

## Requirement
* PHP
* Ngrok (optional)

## Run locally

* Run this command
```sh
$ php -S 0.0.0.0:8000
```

* (optional) if you need to share the website for uploading, you need to install ngrok and run this command.
```sh
$ ngrok http 8000
```
